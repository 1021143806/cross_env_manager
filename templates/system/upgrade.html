API 调用方式 
# 通过 curl 升级（带备注）

  -H "Cookie: session=你的SESSION_COOKIE" \
  -F "file=@upgrade_v2.4.2.zip" \
  -F "remark=修复了xxx问题，新增了xxx功能"
